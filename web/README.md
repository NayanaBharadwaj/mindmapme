# MindMapMe Frontend

This is the frontend application for **MindMapMe**, an AI-powered mind mapping platform that converts unstructured ideas into structured visual knowledge graphs.

The frontend provides the interactive interface for creating, editing, and exploring mind maps while communicating with the ASP.NET Core backend API.

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* shadcn/ui

## Features

* Interactive mind map editor
* Node creation and editing
* Semantic search interface
* AI-assisted question answering
* Responsive UI

## Running Locally

Install dependencies:

```
pnpm install
```

Start the development server:

```
pnpm dev
```

The app will be available at:

```
http://localhost:3000
```

## Backend API

The frontend communicates with the backend API available at:

```
https://mindmapme.onrender.com
```
